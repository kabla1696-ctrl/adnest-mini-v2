# AdNest Mini v2

A lightweight Chrome extension that shows a small sponsored widget with:
- Category-based ad selection
- Frequency control
- Enable/disable toggle
- Basic impression/click stats

## Features
- Manifest V3
- Works on all URLs (`<all_urls>`)
- Configurable categories (`crypto`, `tech`, `finance`)
- Blocked domain support
- Popup stats panel
- Options settings page

## Install (Developer Mode)
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this project folder

## Settings
Go to extension **Details → Extension options**:
- Frequency (minutes)
- Categories (comma-separated)
- Blocked domains (one per line)

## Notes
- This is an opt-in sponsored widget model.
- No deceptive injection behavior is used.

## Version
Current version: `0.0.2`
